Description

The Alcina template is a 3-project combo:
  • entities: ejb3 module containing
    • persistable entities (start set: user, group, iid, wrapped object, datatransform persistence, (*) publication
    • client-server model objects, actions
    • persistence access code
    • handles the following spis: (fill-in, template ann)
  • server: war module containing
    • remote call handler, servlet initialisation and central point
    • server action handlers
  • client: gwt client module containing
    • app module (either inherits clientbase or not)
    • widget image providers
    • (todo: abstract layoutmanager,maincmp, maintabpanel)
