The Dirndl Cookbook

...or how do I wrestle this dang thing into what I want?

The @Directed annotations aim to produce sensible results, via sensible defaults for the most common use cases - but that means the uncommon cases require a little more wrangling:

Common cases

Rendering a java bean (any class with getters)


class Croissant() {
  @Directed
  String filling = "ham n cheese"
}

--> 

<croissant>
  <span>ham n cheese</span>
</croissant>

Transforming the entries of a list

To transform the elements of a list, add a @Directed.TransformElements( <transform-class>) annotation - e.g. to transform List<String> to List<HtmlModel>, add @Directed.TransformElements(LeafModel.HtmlModel.Transform.class)
