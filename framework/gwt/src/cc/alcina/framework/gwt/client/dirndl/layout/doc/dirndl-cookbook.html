The @Directed annotations have aim for the most common use cases, but that means the uncommon cases require a little more wrangling:

Transforming a list

The simplest transform annotation, applied to a list property, transforms the elements, not the containing list. To transform the parent list, you must explicitly set the renderer on the annotation:

			

//transforms each Entry element of list - not what you want (since the renderer is the
// default for a java.util.Collection subtype, DirectedRenderer.Collection)
@Directed.Transform(Tables.Multiple.class)
public List<Entry> getEntries() {
  return this.entries;
}

//This one
@Directed(renderer=DirectedRenderer.TransformRenderer.class)
@Directed.Transform(Tables.Multiple.class)
public List<Entry> getEntries() {
  return this.entries;
}
